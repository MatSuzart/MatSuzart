It all started when I saw my **first script in the RPG Tibia**.  
That was my first contact with **programming logic and automation**, and I immediately fell in love with it.

Soon I started writing my own scripts.  
So if you ever stumbled upon a bot during a hunt back in the day… I'm sorry about that. 😅
